# Inferno: Moral Acoustics

An interactive acoustic mapping of select Circles and Bolgias of Dante's Inferno.

## Files

- `inferno_acoustic_map.html` - Main interactive map
- `index.html` - Entry point (redirects to main map)
- `expanded_2.avif` - Inferno topography image
- `*.m4a` - Audio files for each circle/bolgia

## Usage

Simply open `index.html` or `inferno_acoustic_map.html` in a web browser.
